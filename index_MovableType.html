MovableTypeのインストールメモ

  • 本体 http://www.movabletype.org/
  • 日本語化パッチ http://rebecca.ac/milano/mt/readme/ http://rebecca.ac/milano/mt/archives/000168.html
  • mt.cfgに設定したパラメータ 
     CGIPath http://yusuke.homeip.net:9000/diary/mt/
 StaticWebPath /diary/
 NoHTMLEntities 1 (アンコメント)
 PublishCharset utf8
 #PostgreSQLにつなげるので
 ObjectDriver DBI::postgres
 Database mt
 DBUser postgres

パスワードは mt-db-pass.cgi に書く

  • PostgreSQL でデータベース作成 
     $ createdb -E unicode  mt
 CREATE DATABASE
  • Apacheに配置 
     cp -R MT-2.661-full-lib/* /usr/local/apache2/cgi-bin/
 cd /usr/local/apache2/cgi-bin/
 chown -R nobody *
 chgrp -R nobody *
  • httpd.conf を書き換え 
     ScriptAlias /diary/mt/ "/usr/local/apache2/cgi-bin/"
 <Directory "/usr/local/apache2/diary/mt">
     AllowOverride None
     Options None
     Order allow,deny
     Allow from all
 </Directory>

Project Lead: Yusuke Yamamoto
URL: http://yusuke.homeip.net/komuso/
Description:
Komuso is a very simple MBean server monitoring tool. With Komuso, you can monitor any MBean server including WebLogic Server, Tomcat, JBoss, Oracle Application Server, WebSphere and just JVM locally or remotely. Komuso is unique for its simplicity. Komuso consists of only 150 lines of code. This makes Komuso easier to understand and customize.

All Projects : Log4Twitter (Key: LFT)

Project Lead: Yusuke Yamamoto
URL: http://yusuke.homeip.net/log4twitter/
Description:
Log4Twitter is an Appender plug-in for Log4j and Logback that sends log messages to twitter. With Log4Twitter, you can integrate any application using Log4j or Logback without any modification to the code.

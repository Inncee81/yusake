そば1.1

そばとは

そばは 簡単 かつ 強力な web ベースの HTML エディタです.

何が出来ますか？

左側のフレームで htmlを編集すると、タイプする毎に右側のフレームが更新されどのように表示されるか確認出来ます。
保存->リロード といった操作が必要ないのでCSS 、JavaScript など細かい調整を必要とする部分の開発に便利です。
また編集中のhtmlはクッキーに保存(約4kbまで)されるためブラウザを一旦閉じても続けて編集できます。

私のサイトでもそばを使えますか？

はい。もちろんできます。そばをダウンロードしたらサーバの任意の箇所に配置してください。その後下記の "Sobanize!" ブックマークレットをブックマークしてください。サーバに配置されているhtmlを閲覧中にブックマークレットを起動することでいつでもコンテンツを編集できます。現在のところ編集したコンテンツをサーバに反映させるには手動で保存、アップロードする必要があります。
ブックマークしてください->
ダウンロード->soba1.11_ja.zip

これは Wiki のようなものですか？

いいえ。違います。しかしそうなるかもしれません。注目していてください;)
そばは html をサーバサイドに永続化しないからです。
そのコンセプトは現在のところはどちらかといえば HTML エディタに近いものです。

そばは私のブラウザと互換性がありますか？

そばは JavaScript 技術をベースとしています。
JavaScript が有効になっているブラウザであれば何でもお使い頂けます。
そばは Safari v125.12, FireFox v1.0(Mac/Win) and IE(Macv5.2/Win v6.0SP2). でテストされています。
現在のところ Opera では "Sobanize" 機能を利用するためにはOpera8.0以上のバージョンをお使い頂く必要があります。以前のバージョンはこの機能に必要な XMLHttpRequest APIという機能をサポートしていないからです。
